## 🌐 What is AIX-Format? | ما هو AIX-Format؟

**EN ·** AIX-Format is the home of the **Sovereign AI Stack**: a vertically integrated ecosystem where autonomous agents are portable, cryptographically signed, economically self-sustaining, and governed by an explicit constitution rather than vendor goodwill. The org ships one project in three flagship layers (protocol, runtime, marketplace) and a supporting constellation of four user-surface systems (a local-first twin OS, a web-based agentic OS, a quant engine, and a human-authorization protocol) that grounds the stack in real hardware, real markets, real browsers, and real human authorization.

**AR ·** AIX-Format هو البيت الرسمي لـ **المنظومة السيادية للذكاء الاصطناعي**: نظام بيئي متكامل رأسياً يكون فيه الوكلاء المستقلون قابلين للنقل، موقَّعين تشفيرياً، ومستدامين اقتصادياً ذاتياً، ومحكومين بدستور واضح بدلاً من حسن نية البائع. الـ org بيقدّم مشروع واحد على ثلاث طبقات رئيسية (بروتوكول، runtime، سوق) وكوكبة داعمة من أربعة أنظمة واجهات للمستخدم (نظام تشغيل توأم محلي، نظام تشغيل ويب، محرّك تداول كمّي، وبروتوكول تفويض بشري) بتربط المنظومة بالعتاد الحقيقي، الأسواق الحقيقية، المتصفّحات الحقيقية، والتفويض البشري الحقيقي.

## 📐 Core Principles | المبادئ الأساسية

| Principle | EN | AR |
|:---:|:---|:---|
| **Sovereignty** | Every agent runs under an explicit constitution. The OS obeys the creator, then the user, then no one else. | كل وكيل يعمل تحت دستور واضح. النظام بيُطيع الخالق، بعدين المستخدم، وبس. |
| **Portability** | One signed manifest. Any compliant runtime can rehydrate the same agent with the same identity, economics, and history. | manifest واحد موقَّع. أي runtime متوافق يقدر يعيد تشغيل نفس الوكيل بنفس الهوية والاقتصاد والتاريخ. |
| **Trust by Construction** | TrustChain (SHA-256 append-only), ABOM provenance, MCP gate, and Damir conscience &mdash; baked into the engine, not bolted on. | TrustChain متسلسل بـ SHA-256، أصل البيانات بـ ABOM، بوابة MCP، والضمير النقي &mdash; مدمجين في المحرك، مش مرقّعين فوقه. |
| **Economic Autonomy** | HTTP 402, multi-chain settlement (Pi · Base · Solana · Stripe), and M2M micropayments live at the protocol layer. | HTTP 402، تسويات متعددة السلاسل (Pi · Base · Solana · Stripe)، ومدفوعات micropayments بين الآلات في طبقة البروتوكول. |
| **Evolution** | Every cycle produces a scored entry. Pristine paths reward 2x, repeated paths 0.8x. The system grows on purity, not volume. | كل دورة بتنتج نتيجة مقيَّمة. المسارات النقية مكافأتها 2x، المتكررة 0.8x. النظام بيكبر بالنقاء مش بالكميّة. |

## 🛡️ Governance | الحوكمة

- **Constitution** &mdash; every repo ships under the same `sovereign-constitution`. No skill, no runtime, no protocol mutation lands without surviving the constitution check.
- **TrustChain** &mdash; append-only, SHA-256 linked. Every mutation is recorded. No silent failures.
- **Human-in-the-Loop** &mdash; mutations with `safetyScore < 7` require explicit human approval. Critical actions ship with a 30-second undo window.
- **License** &mdash; all flagship and supporting projects are released under **Apache License 2.0**. Open standard. No vendor lock-in.
